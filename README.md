# mysite1
Blog website with wagtail, django, python

Personal blog developed with Wagtail Content Management System, Django Framework, and Python.

First, from an elevated administrator terminal pip install wagtail.

Once installed, at the command prompt enter wagtail start mysite.

Verify that you are in the directory with manage.py, then enter py manage.py createsuperuser.

Follow prompt by entering name, email address, and password. 

Upon credential aceptance, within same directory as manage.py, input py manage.py runserver.

Server will be located at localhost 127.0.0.1:8000/admin/, enter address within browser search bar.

Navigate to page and enter logging information to begin building webpage.
